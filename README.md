# 倉庫番(後期課題)
DxLibを使用して作成している倉庫番

## 実装内容

* タイトル画面などに情報を書いてユーザが遊べるようにする
* 画像データを使用して、矩形表示を画像表示にする
* キー入力に応じてプレイヤーの向きが変わるようにする
* クリアするためにかかった手数をゲームシーンとリザルトシーンに表示する
